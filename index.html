Firebase Setup Guide
⚡ One-time setup — takes ~10 minutes
After this, your app will sync live across all phones and devices — no re-setup needed.
1 Create your Firebase project
1. Go to console.firebase.google.com
2. Click "Add project"
3. Name it joel-two-army
4. Disable Google Analytics (not needed)
5. Click "Create project"
2 Add a Web App
1. In your project, click the Web icon (</>)
2. Name it Joel Two Army
3. Click Register app
4. Copy the firebaseConfig object shown
5. Paste it in the file where it says PASTE_YOUR_API_KEY etc.
3 Enable Authentication
1. In Firebase Console → Authentication
2. Click "Get started"
3. Enable Email/Password sign-in
4. Create your leader account:
  → Users tab → Add user
  → Email: your email, set a password
4 Create Firestore Database
1. Firebase Console → Firestore Database
2. Click "Create database"
3. Choose Production mode
4. Pick a server location closest to you
5. Click Done
5 Set Firestore Rules
In Firestore → Rules tab, paste this:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }
Click Publish
6 Host on GitHub Pages (free)
1. Go to github.com → sign in / create account
2. Create a new repository called joel-army
3. Upload your HTML file
4. Settings → Pages → Branch: main → Save
5. Your app URL will be:
yourusername.github.io/joel-army

Share this URL with all parents and children — everyone logs in from the same link!
✅ After setup is complete
Open the file, fill in your Firebase config, save it, then upload to GitHub. The setup screen will disappear and your live app will appear.
